# Mutual Fund Analytics & Performance Intelligence Platform

## Project Overview

This project is an end-to-end Mutual Fund Analytics and Performance Intelligence Platform developed as part of the Bluestock Data Analytics Internship Capstone Project.

The platform integrates data engineering, database management, SQL analytics, performance evaluation, risk assessment, and business intelligence dashboards to provide actionable insights into mutual fund performance and investor behavior.

---

## Objectives

- Build a complete ETL pipeline
- Create a centralized SQLite database
- Perform exploratory data analysis (EDA)
- Analyze mutual fund performance
- Evaluate investment risk using advanced metrics
- Develop interactive dashboards
- Generate business insights and recommendations

---

## Data Sources

The project uses the following datasets:

1. Fund Master Data
2. NAV History
3. AUM by Fund House
4. Monthly SIP Inflows
5. Category Inflows
6. Industry Folio Count
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices
